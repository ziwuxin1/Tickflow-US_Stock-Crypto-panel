# Handoff: Followin 数据检索 · AI 智能体（Cyberpunk 2077 主题）

## Overview
一个悬浮在「个股分析」页之上的 **Followin 数据检索 AI 智能体** 弹窗，风格为 AlphaFlow 量化终端的 Cyberpunk 2077 主题（酸性黄主色 + 青色信息色 + 切角/glitch/mono 微文）。核心能力：
- 用户可**自建多个 AI 智能体**（身份：名称 / 角色头衔 / 分组 / 头像色 / 一句话简介），并从工具目录中**勾选每个智能体擅长的技能**。
- 智能体按**分组**（美股组 / 加密货币组 / 新闻组 / 信号策略组 …）在左侧栏罗列，分组可**折叠**（带动画）。
- 选中某智能体 → 进入 TA 的**空白对话**（展示身份 + 简介 + 领域相关的快捷问）。
- 提问后走 **加载中（工具轨迹）→ 流式输出 → 定稿** 三态；回复内容按问题关键词 + 当前智能体身份生成。
- 顶部有**「快速取数 / AI 分析」双模式**紧凑分段控件（影响输入提示、加载表现、底部状态）。
- 弹窗为**可拖动、可缩放**的窗口（顶栏拖动、四边四角缩放），带**字号缩放器**（阅读区 80%–115%）。
- 四周有赛博**装饰边框**（双线 + 发光光条 + 方块/刻度标记 + 竖排读数），底部有**全宽状态条**。

## About the Design Files
本包内的 `.dc.html` 文件是**用 HTML 制作的高保真设计参考稿**（浏览器直接打开预览），**不是可直接照搬的生产代码**。任务是：在目标代码库既有技术栈（React / Vue / SwiftUI / 原生等）中，用其既有模式与组件库**重新实现这些设计**；若尚无技术栈，则为项目选择最合适的框架实现。

> 运行时说明：`.dc.html` 使用一套内部「Design Component」运行时（`support.js`）：模板在 `<x-dc>…</x-dc>` 内，全内联样式，`{{ }}` 为数据插值，`<sc-for>` / `<sc-if>` 为循环/条件；逻辑在文件底部 `<script data-dc-script>` 的 `class Component extends DCLogic`，`renderVals()` 返回模板数据。**落地时无需复刻这套运行时**，按下方文档用目标框架重写即可。

## 布局结构（弹窗）
弹窗为 `position:absolute` 的窗口，**上下两段** flex 列：
1. **内容行**（`flex:1;display:flex;min-height:0`）：左 **历史/智能体侧栏（252px 定宽）** ｜ 右 **主区（flex:1，右内边距 20px 供滚动条避让）**
2. **底部状态条**（全宽，`flex:none`）

四周叠加**装饰边框**（4 条绝对定位 overlay，`pointer-events:none`，z-index 6）；四角 L 形角标（z-index 9）；四边 + 四角 **缩放手柄**（透明，z-index 8/9，`cursor:*-resize`）。

默认尺寸：`w=min(1720, 视口宽-8)`，`h=min(1180, 视口高-8)`，居中；最小 `620×460`；可拖动/缩放并夹取在视口内。

## Interactions & Behavior
- **切标签**：智能体 / 历史，切换列表与主行动按钮/搜索占位
- **选智能体**：设为当前 + `session='agent'` → 主区空态（该智能体身份 + 领域建议卡）；头部徽标、输入提示、报告署名随之更新
- **折叠分组**：点分组头 toggle；组内卡片 `max-height/opacity` 过渡收合，箭头旋转 −90°
- **新建/编辑智能体**：打开编辑器 overlay；改身份字段、点色板、点分组 chip/输入、勾选技能卡（toggle）；`保存` 校验名称非空→写入/更新，新分组自动入列，设为当前并进入其对话；`取消` 返回
- **双模式切换**：快速/AI，联动输入占位、加载表现（工具轨迹 vs slim spinner）、底部文案、加速流式速度
- **发送**（输入回车 / 点发送 / 点快捷问 / 点建议卡）：`pending{loading}` →（AI 1400ms / 快速 550ms 后）`streaming` 逐字打字（`setInterval 18ms`，AI +3/快速 +5 字符）→ `done`。回复文本按问题关键词匹配（英伟达/特斯拉/苹果/指数/ETH/SOL/巨鲸/快讯/宏观/信号/BTC…）+ 署名当前智能体 + 技能数；无匹配给中性判断
- **字号缩放**：`A−/A+` 调 `fontScale`（0.8–1.15，步进 0.1，夹取），作用于对话区 `zoom`；点百分比重置 100%
- **拖动**：按头部（排除 `[data-nodrag]`）移动窗口，夹取在装饰框内
- **缩放**：四边/四角手柄拖拽改尺寸（最小 620×460，夹取视口内）；右下角有斜纹把手
- **自动滚动**：发送/流式/切会话时聊天区滚到底（`_pin` 标志控制；拖动/缩放时不滚）
- **hover**：导航项/卡片/chip/按钮均有描边或加亮反馈

## State Management
`Component` 关键 state：
- `mode`: `'ai' | 'fast'`（双模式，默认来自 `defaultMode` prop）
- `sideTab`: `'agents' | 'history'`
- `panel`: `'chat' | 'edit'`（主区 vs 编辑器 overlay）
- `session`: `'agent' | 'btc' | 'new' | <历史id>`（决定主区渲染分支）
- `activeAgent`: 当前智能体 id
- `agents`: 智能体数组 `{id,name,role,group,color,desc,skills:{toolId:1}}`；`groups`: 分组名数组；`collapsed`: `{组名:bool}`
- `draft` / `editingId`: 编辑器草稿与目标 id
- `pending`: `null | {q, phase:'loading'|'streaming'|'done'}`；`streamText`: 流式文本
- `fontScale`: 阅读区缩放；`win`: `{x,y,w,h}` 窗口位置尺寸
- 拖动/缩放用非 state 实例字段（`_drag/_rz/_bw/_bh`）+ window 事件监听；`_pin` 控制自动滚动

Props（可调 tweak）：`defaultMode`（enum ai/fast）、`showBackdrop`（boolean，默认 false，背景暗化页开关）。
